A simple CLI-based number guessing game. You have to guess a number between 1 and 100.
Project for https://roadmap.sh/projects/number-guessing-game

## Features
- Three difficulty levels:
  - Easy: 10 guesses
  - Medium: 5 guesses
  - Hard: 3 guesses
- Option to quit anytime by typing `quit`, `q`, `x`, or `exit`.
- Hints provided in Easy mode (too high / too low).

## How to Run
Clone the repository and run the script with Python 3:

```bash
git clone https://github.com/datdaboi/number-guessing-game.git
cd number-guessing-game
python game.py
